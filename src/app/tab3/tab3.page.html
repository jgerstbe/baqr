Welcome to baQR!

A modern take on luggage tags!

baQR was born after playing around with QR codes and them displaying contact information as a vCard. Originally the idea was to use the QR code on a business card - but from there the idea to use it as a luggage tag arose.

How to use

baQR can be used differently...

1st: As a one time, self-contained read only tag.

  • Create a baQR-Code.
  • Print it!
  • Profit 🥳

2nd: As a cloud connected self-updating tag.

  • Create a baQR-Code.
  • Print it!
  • Profit 🥳
  • Update it via baQR if needed.
  • ... more profit 😁
